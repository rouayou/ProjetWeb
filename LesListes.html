Comment créer une liste en HTML5 ?

Les listes HTML sont prévues pour dresser des énumérations possédant une valeur sémantique.Les balises <OL>, <UL>, <UL> déterminent 3 type des listes ( ordonnée, non ordonnée et définition de liste ). On associe à ces balises, une autre balise <LI> … </LI> précise quels sont les différents éléments de la liste.

  • Liste ordonnée :

    Les éléments de la liste sont numérotés et décalés vers la droite. La syntaxe de cette liste est la suivante :

    <OL> <LI> … </LI> <LI> … </LI> </OL>

  • Liste nonordonnée :

    Les éléments de la liste sont décalés vers la droite et précédés d'une puce.

    <UL> <LI> … </LI> <LI> … </LI> </UL>

  • Liste de définition :

    Les éléments de la liste, non décalé, sont précédés d'une puce

    <DL> <DD> … </DD><DT> … </DT> <DD> … </DD>><DT> … </DT> </DL>

    Les glossaires comprennent une liste d'éléments. Les éléments d'un glossaire sont constitués par une expression à définir ( se mettre entre <DD> et </DD> ) et la définition qui le concerne ( se mettre entre <DT> et </DT> ).

  • Le nombre d'éléments dans la liste est au choix.
