  • Les attributs

    Les attributs sont un peu les options des balises. Ils viennent les compléter pour donner des informations supplémentaires. L'attribut se place après le nom de la balise ouvrante et a le plus souvent une valeur.

    On ne met les attributs que dans la balise ouvrante et pas dans la balise fermante. En d'autre terme la définition complète d'une balise se fait comme suit :

    <nom_balise attribut1="valeur1" attribut2="valeur2"...attributN="valeurN" > Contenu de la balise </nom_balise>

  • Les attributs globaux

    Les attributs globaux sont des attributs qui s’appliquent à quasiment tous les éléments ( balises ) HTML. Ces attributs sont :

    Tableau 2 : Les attributs globaux
    L'attribut Utilisation/Signification
    id Sert à identifier un élément de manière unique en lui donnant un nom ( pour lui attribuer un style ou pour y faire référence sans ambiguïté dans un script )
    class contient le nom d’une classe CSS qui contient des définitions de styles ( pour affecter ponctuellement des styles à un élément )
    style Permet de définir un style localement pour un élément donné ( déconseillé au profit des styles CSS permettant une maintenance plus facile )

  • Les attributs de gestion d’événements

    Ces attributs permettent de gérer les événements dont un élément peut être le siège et qui sont créés par l’utilisateur. Leur contenu est un script écrit en général en langage JavaScript

    Tableau 3 : Les attributs de gestion d’événements
    L'attribut Utilisation/Signification
    onmouseover Le curseur de la souris est au-dessus de la zone de l’élément
    onclick Clic sur le contenu de l’élément
    ondblclick Double-clic sur le contenu de l’élément
