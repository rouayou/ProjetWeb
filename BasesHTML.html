Les bases du langage HTML 5 :

  • Les balises et leurs attributs

    Les pages HTML sont remplies de ce qu'on appelle des balises. Celles-ci sont invisibles à l'écran pour vos visiteurs, mais elles permettent à l'ordinateur de comprendre ce qu'il doit afficher.

    Les balises se repèrent facilement. Elles sont entourées de « chevrons », comme ceci : <balise> contenu de la balise </balise>.

    Tout ce qui est en dehors des symboles < et > est du texte qui sera affiché sur la page Web.

  • Règles de base de HTML 5 :

    Un document HTML doit respecter certaines règles simples:

    1. Les éléments ne doivent pas se chevaucher et donc obéir au principe premier ouvert, dernier fermé.
    2. Les éléments et les attributs sont insensibles à la casse. Il est cependant préférable de tout mettre en minuscules.
    3. Les éléments non vides doivent avoir une balise d’ouverture et une balise de fermeture.
    4. Les éléments vides ne comportent qu’une seule balise.
    5. Tous les attributs doivent avoir une valeur incluse entre guillemets (" ").
    6. Les différents attributs du même élément doivent être séparés par au moins un espace.
    7. Une valeur doit être donnée à tous les attributs utilisés, y compris à ceux dont la valeur est unique.

  • Minimum d'une page HTML 5 :

    <!DOCTYPE html> <html> <head> <!--Entête de la page--> </head> <body> <!--Contenu de la page--> </body> </html>
    <!DOCTYPE html> :
    La déclaration DOCTYPE précise le type de document qui va être créé.

    <html> :
    C'est l’élément racine du document HTML.

    <head> :
    Indique l’en-tête du document.

    <body> :
    Indique le corps du document.

  • Les éléments de la balise <head> :

    L’élément <head> permet une description de la page et englobe un certain nombre d’informations dites métadonnées contenues dans six éléments différents: <base/>, <link/>, <meta/>, <script>, <style> et <title> .

    Son contenu n'est pas affiché par le navigateur ( Sauf l'élément title ).

    Tableau 1 : Les éléments de la balise <head>
    La balise Utilisation
    <base/> Base d’adresses par défaut et de cible pour tous les liens du document
    <link/> Relations vers des ressources externes
    <meta/> Méta-informations ( auteur, description, encodage des caractères...etc ).
    <script> Scripts embarqués dans le document
    <style> Styles CSS embarqués dans le document
    <title> Titre du document

  • Les éléments de la balise <body> :

    L’élément <body> est le conteneur de l’ensemble des éléments textuels et graphiques d’une page web. Dans leur quasi-totalité, les éléments disponibles en HTML ont en commun un ensemble d’attributs ayant chacun le même rôle. Ces attributs se répartissent en deux catégories:

    Les attributs globaux (ou noyau):
    Ils s’appliquent à quasiment tous les éléments.

    Les attributs de gestion d’événements :
    Ces attributs permettent de gérer les événements.

Remarques :

  • On doit fermer toutes les balises.
  • Pas de chevauchement entre les balise ( La plus dernière balise ouvert est la première fermée ).
  • Le langage HTML ne prend pas en considération la casse.
  • Ce qui est compris entre <!-- et --> représente un commentaire en HTML.
